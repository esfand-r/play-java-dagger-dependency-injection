Overview

This example demonstrates usage of Dagger for dependency injection in Play framework.

Run the App

Run the App: (./activator run) Dagger is configured to work with Play in Global.java Configuration of Dependency Injection can be found at DaggerConfig.java

Test

Run the Tests: (sbt test) Please see TestConfiguration. This is where it is configured so the tests can use the service Sample test is at RandomServiceTest.java
